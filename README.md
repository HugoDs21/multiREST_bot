# multiREST_bot
Discord bot for Multirest papas

# Installation
 -  Clone rep
 -  Create a `.env` file
 -  Get your Discord API Token from https://discord.com/developers/applications (more documentation [here](https://discord.com/developers/docs/topics/oauth2))
 -  `docker-compose up -d`
